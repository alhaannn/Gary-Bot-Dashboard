# GaryBot Dashboard Setup

These are the environment variables required when deploying this dashboard to Vercel (or any other provider). In Vercel, paste these into the **Environment Variables** section:

```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project-id.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key_here
NEXT_PUBLIC_STARTING_BALANCE=2500
```
